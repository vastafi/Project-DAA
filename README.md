## Installation

1. Clone the repository:
    ```sh
    git https://github.com/vastafi/Project-DAA
    cd DrugSystem
    ```

2. Install Maven (if not already installed):
    ```sh
    brew install maven
    ```

3. Verify Maven installation:
    ```sh
    mvn -version
    ```
